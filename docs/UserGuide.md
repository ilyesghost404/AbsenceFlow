# User Guide

## 1. Dashboard

The Dashboard provides an overview of key metrics, including total employees, total absences, validated absences, and attendance rate.

## 2. Employees

### Adding an Employee
1. Click **Add Employee** button
2. Fill out the form with employee details
3. Click **Save**

### Editing/Deleting an Employee
- Use the Edit/Delete buttons on each employee row

## 3. Attendance

The Attendance section has two tabs: **Presence** and **Absences**.

### Presence Tab
- Displays today's attendance for all employees
- Use **Check In** and **Check Out** buttons to record attendance

### Absences Tab
- Displays all absence records
- Absences are automatically created for employees who don't check in by end of day
- Edit or delete existing absences, or validate pending absences

## 4. Holidays

### Adding a Public Holiday
1. Click **Add Holiday**
2. Select the date and enter a name
3. Click **Save**
- Holidays are automatically excluded from working day calculations.

## 5. Reports

### Generating a Monthly Report
1. Go to Reports
2. Select year and month
3. Click **Generate Excel Report**
- The Excel file will download automatically with all attendance data for the month.
